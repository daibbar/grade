## Follow the instructions to use and test the application

1. **git clone the repository in your local machine**
2. **run the following commands one by one inside of your repository**:  
    ```bash
    python3 -m venv venv
    source venv/bin/acitvate
    pip install -r requirements.txt
    ```
3. **to test the application, run the following command and visit localhost:8501 in your browser**:
    ```bash
    streamlit run app.py
    ```
